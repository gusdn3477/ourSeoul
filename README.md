## 우리의 서울

### 소개
코딩온 파이썬 웹 과정 4팀(박현우, 이유영, 이성민)의 프로젝트 결과물 '우리의 서울'입니다.  
서울에 대한 소개와 선정된 명소들에 대한 거리뷰, 키워드 검색등을 제공하는 웹 사이트입니다.

### 사용한 것들
- Front-end : HTML, CSS, JavaScript, BootStrap  
- Back-end : Django  
- 배포 : Naver Cloud Platform
